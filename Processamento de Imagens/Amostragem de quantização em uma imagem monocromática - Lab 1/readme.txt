Utilize as seguintes instruções para execução do programa:

    python3 lab1.py -i <imagem> -p <porcentagem> -t <tecnica> -q <quantizacao>

- lab1.py: nome do programa
- imagem: nome do Arquivo
- porcentagem: valor em formato decimal
- tecnica: entrar com a string moda, media ou mediana
- quantizacao: valor inteiro entre 2 a 256

Para exibir essas informações no terminal, basta entrar com:

    python3 lab1.py -h


É obrigatório inserir os quatro parâmetros descritos acima. Se não, é retornado erro.
O arquivo de saída é nomeado como 'saida.png'
